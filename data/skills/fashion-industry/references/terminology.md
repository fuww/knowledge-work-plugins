# Fashion Industry Terminology

Comprehensive glossary of fashion industry terms, roles, materials, and concepts for accurate communication and analysis.

## Season and Collection Terms

### Season Abbreviations

| Term | Meaning | Example |
|------|---------|---------|
| SS / S/S | Spring/Summer | SS26 = Spring/Summer 2026 |
| AW / A/W | Autumn/Winter | AW25 = Autumn/Winter 2025 |
| FW / F/W | Fall/Winter (US usage) | FW25 = Fall/Winter 2025 |
| PF | Pre-Fall | Pre-Fall 2025 |
| RS | Resort | Resort 2025 |
| CR | Cruise | Cruise 2025 |
| HC | Haute Couture | HC SS25 |

### Collection Types

| Term | Definition |
|------|------------|
| Main collection | Primary seasonal collection (SS or AW) |
| Pre-collection | Inter-season collection (Pre-Fall, Resort/Cruise) |
| Capsule collection | Small, focused collection (often collaborative) |
| Limited edition | Restricted quantity release |
| Core/Continuity | Year-round staple items |
| Carry-over | Items continued from previous season |
| NOS | Never Out of Stock (core items) |

### Presentation Formats

| Term | Definition |
|------|------------|
| Runway show | Traditional catwalk presentation |
| Presentation | Static or semi-static display of collection |
| Lookbook | Photographic collection documentation |
| Film/Video | Digital presentation format |
| Showroom | Private viewing for buyers/press |
| Trunk show | Designer appearance at retail location |
| Pop-up | Temporary retail or exhibition space |

## Garment and Product Terms

### Garment Categories

| Term | Definition |
|------|------------|
| RTW / Ready-to-wear | Mass-produced fashion (pret-a-porter) |
| Haute couture | Custom-made high fashion (protected term in France) |
| Bespoke | Custom-made to individual measurements |
| MTM / Made-to-measure | Customized from existing pattern |
| Demi-couture | High-quality RTW with couture details |

### Product Classification

| Term | Definition |
|------|------------|
| Look / Exit | Single outfit in runway show |
| Style / SKU | Individual product identifier |
| Colorway | Color variation of same style |
| Size run | Range of sizes available |
| MOQ | Minimum Order Quantity |
| Assortment | Product selection for retail |
| Buy | Retailer's order from brand |

### Price Points

| Term | Definition | Examples |
|------|------------|----------|
| Luxury | Highest price tier | Chanel, Hermes, Louis Vuitton |
| Premium / Bridge | Mid-luxury | Tory Burch, Coach, Kate Spade |
| Contemporary | Upper-middle market | Reiss, Sandro, Theory |
| Mid-market | Middle price tier | Massimo Dutti, COS |
| Fast fashion | Affordable, trend-driven | Zara, H&M, Primark |
| Value / Budget | Lowest price tier | Shein, Boohoo |

## Materials and Textiles

### Natural Fibers

| Fiber | Properties | Common Uses |
|-------|------------|-------------|
| Cotton | Breathable, soft, absorbent | Basics, casualwear, denim |
| Wool | Warm, resilient, moisture-wicking | Suiting, knitwear, outerwear |
| Silk | Lustrous, smooth, temperature-regulating | Luxury, evening wear, blouses |
| Linen | Cool, breathable, textured | Summer wear, resort |
| Cashmere | Soft, warm, lightweight | Luxury knitwear |
| Mohair | Lustrous, durable, warm | Knitwear, suiting |
| Alpaca | Warm, hypoallergenic, soft | Knitwear, outerwear |

### Synthetic and Man-Made Fibers

| Fiber | Properties | Common Uses |
|-------|------------|-------------|
| Polyester | Durable, wrinkle-resistant, quick-dry | Sportswear, linings, blends |
| Nylon | Strong, elastic, lightweight | Activewear, hosiery, outerwear |
| Viscose/Rayon | Drapes well, breathable | Dresses, blouses, linings |
| Lyocell/Tencel | Sustainable, soft, breathable | Sustainable fashion |
| Modal | Soft, absorbent, stable | Underwear, loungewear |
| Spandex/Elastane | Stretchy, shape-retaining | Activewear, denim, fitted garments |
| Acrylic | Wool-like, affordable | Knitwear, blankets |

### Sustainable Materials

| Material | Description |
|----------|-------------|
| Organic cotton | Grown without synthetic pesticides |
| Recycled polyester (rPET) | Made from post-consumer plastic |
| Econyl | Regenerated nylon from waste |
| Pinatex | Leather alternative from pineapple leaves |
| Mylo | Mushroom-based leather alternative |
| GOTS-certified | Global Organic Textile Standard |
| OEKO-TEX | Tested for harmful substances |

### Leather and Alternatives

| Term | Definition |
|------|------------|
| Full grain | Highest quality, unaltered surface |
| Top grain | Sanded/buffed surface |
| Genuine leather | Lower quality, multiple layers |
| Bonded leather | Leather fibers bonded with adhesive |
| Vegan leather | Plant-based or synthetic alternative |
| PU leather | Polyurethane-coated fabric |

## Sustainability Terms

### Certifications and Standards

| Certification | Focus |
|---------------|-------|
| GOTS | Organic textiles |
| OEKO-TEX | Chemical safety |
| Bluesign | Environmental impact |
| Fair Trade | Worker welfare |
| B Corp | Overall corporate responsibility |
| SA8000 | Social accountability |
| ISO 14001 | Environmental management |

### Sustainability Concepts

| Term | Definition |
|------|------------|
| Circular fashion | Design for longevity, reuse, recycling |
| Closed-loop | Waste materials recycled into new products |
| Upcycling | Transforming waste into higher-value products |
| Deadstock | Unused fabric or surplus inventory |
| Zero waste | Design minimizing material waste |
| Carbon neutral | Net-zero carbon emissions |
| Scope 1/2/3 | Emissions categories (direct, energy, supply chain) |
| LCA | Life Cycle Assessment |
| EPR | Extended Producer Responsibility |
| Greenwashing | Misleading environmental claims |

### Supply Chain Terms

| Term | Definition |
|------|------------|
| Tier 1 | Final assembly factories |
| Tier 2 | Component suppliers (fabric mills, trim) |
| Tier 3 | Raw material processors |
| Tier 4 | Raw material producers |
| Traceability | Ability to track materials through supply chain |
| Transparency | Disclosure of supply chain information |
| Due diligence | Supply chain risk assessment |

## Retail and Business Terms

### Retail Metrics

| Term | Definition |
|------|------------|
| Sell-through | Percentage of inventory sold |
| Markdown | Price reduction |
| Full price | Non-discounted price |
| ASP | Average Selling Price |
| UPT | Units Per Transaction |
| Conversion | Visitors who make purchase |
| LFL / Comp sales | Like-for-like/comparable sales |
| Gross margin | Revenue minus COGS |

### Distribution Models

| Term | Definition |
|------|------------|
| DTC | Direct-to-Consumer |
| Wholesale | B2B sales to retailers |
| Concession | Brand-operated space within store |
| Franchise | Licensed retail operation |
| Department store | Multi-brand retail (Selfridges, Nordstrom) |
| Specialty retail | Category-focused retailer |
| E-commerce | Online sales channel |
| Marketplace | Platform hosting multiple sellers |

### Inventory Terms

| Term | Definition |
|------|------------|
| OTB | Open-to-Buy (buying budget) |
| Lead time | Time from order to delivery |
| Reorder | Additional inventory purchase |
| Replenishment | Restocking sold items |
| Markdown reserve | Budget for price reductions |
| Weeks of cover | Inventory duration at current sales |
| Dead stock | Unsold inventory |
| Overstock | Excess inventory |
